Obrigado por querer contribuir com este projeto! Seu apoio é muito importante para melhorar o conteúdo e a experiência de aprendizado.

## Como contribuir

Você pode contribuir de várias formas:

- Melhorando a documentação
- Corrigindo erros de texto ou formatação
- Sugerindo novas ideias ou conteúdos
- Reportando problemas

## Passos para contribuir

1. Faça um fork do repositório.
2. Crie uma branch para a sua alteração.
3. Faça as mudanças de forma clara e organizada.
4. Commit com uma mensagem descritiva.
5. Abra um pull request explicando o que foi alterado.

## Regras básicas

- Seja respeitoso e acolhedor com os demais participantes.
- Mantenha o conteúdo claro e objetivo.
- Evite alterações desnecessárias.
- Teste as mudanças sempre que possível.

## Relato de problemas

Se encontrar algum problema, abra uma issue descrevendo:

- O que aconteceu
- Quais passos foram realizados
- Qual resultado esperado
- Qual resultado real

## Agradecimento

Agradecemos por dedicar seu tempo para contribuir com este projeto e ajudar a comunidade a crescer.
