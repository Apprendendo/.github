## Como publicar o site?

O site pode ser publicado automaticamente no GitHub Pages a partir da branch principal.

## Como adicionar novas páginas?

Crie um novo arquivo Markdown dentro da pasta [docs](./) e adicione um link na navegação principal.

## Como testar localmente?

Use o comando abaixo para visualizar as alterações antes de publicar:

```bash
bundle exec jekyll serve
```
