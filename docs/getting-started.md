## Pré-requisitos

- Ruby
- Bundler
- Git

## Executar localmente

```bash
gem install bundler
bundle install
bundle exec jekyll serve
```

Depois disso, abra o endereço mostrado no terminal no navegador.

## Estrutura do site

- A página inicial está em [index.md](../index.md)
- As páginas de documentação estão na pasta [docs](./)
